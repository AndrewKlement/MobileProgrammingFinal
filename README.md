# MobileProgrammingFinal
This app is a simple **Android project** created to demonstrate how to fetch and display data from an API, parse JSON, and present the results using a **RecyclerView**. Each list item can be tapped to view detailed information on a separate screen.

## API Used
https://jsonplaceholder.typicode.com/todos

## How to Run
1. Clone the project from GitHub using:
``https://github.com/AndrewKlement/MobileProgrammingFinal.git``
2. Open the project in **Android Studio**
3. Run the app on an emulator or physical device
